# 05 - Experimentacion general

Objetivo: dejar auditadas las pruebas laterales, descartes y alternativas que no
deben dominar la narrativa principal de entrega.

Notebook principal:

- `01_experimentos_generales.ipynb`

Esta carpeta agrupa:

- pruebas que ayudaron a entender el problema pero no entran al modelo final;
- candidatos que mejoraron validacion local pero no Kaggle;
- variantes demasiado complejas para la defensa;
- experimentos superados por el sistema final.

Conclusion esperada: hubo exploracion amplia, pero la entrega se mantiene
enfocada en un sistema simple de tres bloques conceptuales.
